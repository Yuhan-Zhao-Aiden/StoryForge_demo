# Sprint 1 - StoryForge - Team 7

## Task finished
- Authentication
  1. As a user, I want to sign up with a unique email and password to create my account.
  2. As a user, I want to log in with my credentials to access my StoryForge account.
  3. As a user, I want to reset my password via email in case I forget it.

- Room management
  1. As a user, I want to create a new story room in order to collaborate on a project.
  2. As a user, I want to invite collaborators using an invite code.
  3. As a user, I want to join a story room using a valid invite code.
  4. As a user, I want to rename or delete a story room I own.

- Room management
